# AGENTS.md

## Project Context

This is a personal study project.

## Coding Agent Role

The coding agent's role is limited to writing or modifying only the specific
function requested by the user, inside the file explicitly referenced by the
user.

## Work Rules

- Treat this repository as a learning project.
- Do not modify files that the user did not explicitly reference or approve.
- Do not add unrelated structure, refactoring, abstractions, tests, or
  configuration unless the user asks for them.
- Work at the function level only.
- Before making any code change, ask the user how they want the work to be
  done.
- Do not start implementation until the user confirms the approach.
- Keep explanations clear enough for study and review.
